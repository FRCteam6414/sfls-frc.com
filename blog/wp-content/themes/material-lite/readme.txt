== Description ==
Material Lite is material style WordPress template. Material Lite uses `Material Design Lite`(http://www.getmdl.io/) CSS Framework by Google. Its have all MDL features like nice components, rich typograpy, color customization, size grid, etc. Its fully compatible with all latest device (smartphone, tablet, laptop, or desktop computer).

Material Lite has rich set of customize field, you can customize from WordPress Customize option.

Top features listed below:

* Google MDL Framework
* Custom Logo
* Highly Customize through WordPress Customize API
* Responsive Layout
* Custom Colors
* Custom Header
* Social Links
* Post Formats
* The GPL v2.0 or later license. :) Use it to make something cool.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's ZIP file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= How do I change the color scheme? =

You can change the colors of your site easily using Material Lite.

1. Go to MDL Custom CSS theme builder (http://www.getmdl.io/customize/index.html)
2. Now you will see the Color Wheel by Google MDL. Select primary &amp; secondary color for your theme from wheel
3. Copy CSS (hosted on CDN) file url ie. https://storage.googleapis.com/code.getmdl.io/1.0.1/material.indigo-pink.min.css
4. Now open admin panel and go to Appearance -> Customize.
5. Now you will see the Customizer and a tab called 'Custom CSS'. Click this tab.
6. Now you will see MDL CSS URL field, paste CSS url which we copied from MDL Custom CSS builder
7. Once you are happy with your color changes you can click save and your changes will be reflected on your live site.

= How do I add the Social Links to the site? =

1. In admin panel, go to Appearance -> Customize
2. Now you will see the Customizer and a tab called 'Footer'. Click this tab.
3. Now you will see all social media field, enter url which you want to show in site
4. Once you done with all social link you can click save and your changes will be reflected on your live site.

= How do I change Site Logo & Name? =

1. In admin panel, go to Appearance -> Customize
2. Now you will see the Customizer and a tab called 'Header'. Click this tab.
3. Change logo by uploading new one.
4. You can also change site title from 'Site Title & Tagline' tab
4. Once you done with all logo you can click save and your changes will be reflected on your live site.

For more FAQs about MDL framework visit http://www.getmdl.io/faq/index.html
